CMT307 Applied machine learning coursework part 2 -

The coursework file for sentiment analysis of movie reviews on IMDB.
System requirements - 
. Internet connection.
. Jupyter Notebook/Google Collab.
. Import required packages

How to run the code - 

The code is developed as 'sentiment_analysis.ipynb' with Jupyter Notebook in google colab. The code is expected to run with Google Colab and jupyter notebook environment. 
The dataset is in the same master directory and does not need to be downloaded


Performance Measurements- 

The whole process achieves classification precision of 83.9%, recall of 83.7%, f1 score of 83.7%, and accuracy of 83.7% on the IMDb dataset.
